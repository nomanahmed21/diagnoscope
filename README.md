---

## 🚀 Features
- 📊 **Data Collection** – User inputs medical data via frontend forms  
- 🗄 **Database Integration** – Stores patient records in MongoDB  
- 🧠 **ML Model** – Predicts diabetes likelihood from input data  
- ⚡ **Real-Time Graphs** – Visualizes reports and prediction data in live charts  
- 🔄 **API Endpoints** – Secure communication between frontend, backend, and ML service  
- 🧹 **Modular Design** – Clean separation of concerns (frontend, backend, ML model)
